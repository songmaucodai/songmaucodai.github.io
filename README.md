Đây là mã nguồn cho trang chủ cá nhân (Landing Page) tại địa chỉ **[songmaucodai.online](https://songmaucodai.online)**.

Trang web đóng vai trò là "Tổng hành dinh" (Central Hub), giúp điều hướng người dùng đến các dự án, công cụ và thông tin liên hệ của tôi.

## Tính năng (Features)

* **Glassmorphism UI:** Giao diện hiệu ứng kính mờ hiện đại, tối giản.
* **Responsive Design:** Hiển thị tốt trên cả máy tính và điện thoại di động.
* **Smart Navigation:** Liên kết trực tiếp đến các Sub-pages (Info, Pay Stub Tool) và External Repositories.
* **Interactive Donate:** Nút ủng hộ dạng Accordion (mở rộng/thu gọn) tinh tế.

## Công nghệ sử dụng

* **HTML5 / CSS3:** Không sử dụng framework nặng nề (như Bootstrap), code thuần để tối ưu tốc độ tải.
* **Google Fonts:** Sử dụng font `Inter` cho vẻ ngoài hiện đại.
* **FontAwesome 6:** Sử dụng cho các biểu tượng (Icons).

## Cấu trúc dự án

- `index.html`: File chính chứa toàn bộ giao diện và logic.
- `CNAME`: File cấu hình tên miền tùy chỉnh (songmaucodai.online).
